Fast Health Interoperability Resources (FHIR) v<%version%>

Warning: FHIR is a draft specification that is still undergoing development prior to balloting as a full HL7 standard

Fast Healthcare Interoperability Resources (FHIR) defines a set of "resources" for health which represent granular clinical concepts that can be exchanged in order to quickly and effectively solve problems in healthcare and related process. This document is the full FHIR specification (version <%version%>).

The FHIR home page is http://hl7.org/fhir, and has links to other representations and versions of FHIR, along with references to various implementation resources.
